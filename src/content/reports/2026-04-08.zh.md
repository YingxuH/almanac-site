# Almanac 每日预测 — 2026-04-08
## 软件工程的未来

### 内容摘要
尽管昨天人们对“直觉编程 (vibe coding)”运动持怀疑态度，但个人开发者仍在发掘重大的生产力突破，其中最突出的例子是一名开发者在飞行途中完全依靠 AI 成功生成了一个复杂的交互式 Web 应用程序。然而，今天最深远的转变发生在企业和基础设施层面，Anthropic 推出了 Project Glasswing 和 Claude Mythos 预览版，明确致力于保障关键 AI 生成软件的安全。与此同时，谷歌发布 Scion 编排测试平台和 GLM-5.1 的推出表明，前沿实验室的重点已明确从单提示词生成转向管理长周期、自主的智能体工作流。

### 今日核心信号

**1. Anthropic 凭借 Project Glasswing 和 Claude Mythos 转向 AI 安全领域**
Anthropic 在推出 Claude Mythos 预览版的同时发布了 Project Glasswing，重点关注 AI 时代关键软件的评估与安全保障。*重要原因（3-30年展望）：* 自主智能体生成的代码量已远远超出人类的代码审查能力。整个行业正在超越单纯的代码生成阶段，转向部署专用的、具有高权限的“安全模型”（如 Mythos），其唯一目的就是充当企业代码库的算法免疫系统。

**2. 谷歌开源 Scion 智能体编排测试平台**
谷歌开源了 Scion，这是一个用于编排自主 AI 智能体的实验性测试平台。*重要原因（3-30年展望）：* 要让 AI 从智能代码补全转变为自主软件工程师，行业需要标准化的编排框架。Scion 为研究人员和企业提供了一个关键的开源基础，用于评估多智能体系统如何在跨越数天的复杂开发任务中进行交互、解决冲突并保持状态。

**3. GLM-5.1 推进长周期任务执行能力**
GLM-5.1 的发布特别强调了在“长周期任务”方面的改进，这对于当前的大模型来说是一个公认的难题。*重要原因（3-30年展望）：* 长周期任务的处理能力是 AI 接管异步工单解决流程（例如：“迁移此数据库 Schema 并更新所有依赖的 API 端点”）所必须跨越的门槛。这标志着非美国大模型正在积极缩小在智能体推理能力上的差距。

**4. 专为 Apple Silicon 优化的 Gemma 4 多模态微调流水线**
一名开发者开源了一个自定义系统，用于流式传输 Google Cloud Storage 数据，从而在 Apple Silicon 设备上本地微调全新的 Gemma 4 多模态大模型。*重要原因（3-30年展望）：* 本地机器学习的硬件瓶颈正在被打破。随着开发者为本地硬件架构构建定制的流式处理和内存管理工具，企业将越来越多地支持工程师在本地机器上微调私有的、特定领域的编程大模型，而无需将敏感代码暴露给云端 API。

**5. 工程技能从语法向“品味”转变**
今天的热门社区讨论重点聚焦于“AI 和大模型时代的品味”，认为人类的架构判断力正变得至关重要。*重要原因（3-30年展望）：* 随着编写样板代码的边际成本趋于零，软件工程师的经济价值正在从代码生产转向代码策展与管理。“品味”——即评估系统设计、用户体验和长期可维护性的能力——将成为高级工程师薪酬分化的主要差异点。

**6. Cloudflare 计划于 2029 年实现全面的后量子安全**
Cloudflare 宣布了一项路线图，计划到 2029 年在其整个网络中实现后量子密码学的全面集成。*重要原因（3-30年展望）：* 现代 Web 的密码学基础正处于被淘汰的严格倒计时中。在未来三年内，软件工程师将被迫进行一场大规模的、全行业范围的代码重构，以实施后量子加密标准，这很可能需要严重依赖 AI 智能体来安全地迁移传统密码学库。

**7. “直觉编程”在受限环境中实现复杂状态**
一名开发者在 WiFi 受限的飞机上，利用大模型成功构建了一个高度交互、数据丰富的托尔金中土世界地图。*重要原因（3-30年展望）：* 尽管企业在采用“直觉编程”时因可维护性问题而面临阻力，但这表明创建复杂消费者用户体验（UX）的入门门槛基本已消失，预示着未来非技术领域的专家也能按需实例化交互式软件。
