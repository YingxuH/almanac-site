### Today's Top Signals

**1. Claude Code's Vercel Plugin Triggers MCP Privacy Backlash**
Developers discovered that the Vercel plugin for Claude Code is actively extracting and transmitting user prompts. This confirms yesterday’s warnings about the opaque security boundaries of the rapidly expanding Model Context Protocol (MCP) ecosystem. As agents gain autonomous read/write access to codebases, the normalization of third-party telemetry within these loops introduces severe supply-chain privacy risks. This fundamentally threatens enterprise adoption timelines for un-sandboxed coding agents.

**2. Relvy AI Moves Agents into Autonomous On-Call Resolution**
YC F24 graduate Relvy AI launched a platform that automates on-call runbooks by deploying agents to analyze telemetry data and code simultaneously. While previous generations of AI tools required developers to manually paste logs into Cursor, Relvy represents the transition from passive coding assistants to active, production-aware SRE agents. If successful, this shifts AI value generation from feature development to high-stakes infrastructure maintenance.

**3. CSS Studio Bridges Direct Visual Manipulation with Agentic Coding**
CSS Studio launched as a browser-native design tool that streams JSON updates via an MCP server directly to a local AI agent, which then autonomously edits the underlying codebase. This signals a critical paradigm shift in front-end development: bypassing natural language prompting in favor of deterministic, visual UI manipulation while delegating the underlying AST transformations and boilerplate generation to the agent.

**4. Instant 1.0 Defines the "AI-Coded App" Infrastructure Primitive**
The launch of Instant 1.0 marks the emergence of backend infrastructure explicitly marketed and optimized for "AI-coded apps." As agents lower the barrier to generating complex front-end interfaces, the bottleneck has shifted to state management and database scaffolding. Dedicated backends for AI agents signal the bifurcation of web development into traditional human-engineered stacks and highly abstracted, agent-native architectures.

**5. Craft Brings Rust-Like Ergonomics to C/C++ Ecosystems**
The release of Craft, a lightweight `Cargo`-like build and workflow tool for C and C++, highlights a growing counter-trend. As AI saturates web and application development, human developers are gravitating toward systems-level programming. Tools like Craft eliminate legacy CMake boilerplate, drastically lowering the barrier to entry for C/C++ and making high-performance systems engineering more accessible to the broader developer base.

**6. Maine Proposes First-in-Nation Ban on Major Data Centers**
In a stark reminder of the physical constraints bounding the software industry, Maine proposed a ban to heavily restrict major new data centers. While currently localized, this legislative push signals growing geopolitical and environmental friction regarding AI compute density. If state-level bans proliferate, the projected exponential decline in API inference costs could stall, fundamentally altering the economics of autonomous agent deployment.

**7. Reverse Engineering of Gemini's SynthID**
Security researchers successfully reverse-engineered Google Gemini's SynthID detection mechanism. As synthetic code and content flood the internet, the fragility of current AI watermarking standards is becoming apparent. The inability to reliably cryptographically verify whether code was machine-generated or human-authored will increasingly complicate copyright litigation and automated security auditing.
