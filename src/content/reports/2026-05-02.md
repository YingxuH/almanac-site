### Today's Top Signals

**1. Infrastructure Under Siege: Ubuntu Cyberattacks and AWS Physical Damage**
A coordinated, cross-border cyberattack has taken critical Ubuntu server infrastructure offline for over a day, simultaneously overlapping with reports that AWS is halting billing for Middle East customers due to severe drone strike damage to physical data centers. *Why it matters:* The 3-to-10 year outlook for software engineering is increasingly defined not just by application logic, but by extreme infrastructure fragility. **2. The Rejection of "Black Box" AI: Adam's CAD Harness**
Startup Adam released a new AI harness that integrates directly into Autodesk Fusion and Onshape, explicitly pivoting away from prompt-to-3D-model "black boxes." Instead, the AI agentically edits the user's existing feature tree with full visibility. *Why it matters:* This directly reflects today's positive sentiment spike (+0.30) in AI Coding. Professional engineers refuse to adopt AI that obscures the underlying architecture. The next decade of enterprise AI tools will strictly require transparent, deterministic manipulation of existing application states rather than generating opaque, finalized outputs.

**3. Microsoft Releases Lib0xc for Safer Systems Programming**
Microsoft published Lib0xc, a suite of C standard library-adjacent APIs designed specifically to encourage safer systems programming practices. *Why it matters:* Despite massive advancements in AI code generation, the industry recognizes that low-level systems logic (C, C++) remains incredibly brittle. This release indicates that for the next 10-15 years, securing legacy and low-level code will require structural API guardrails, rather than relying solely on AI to write memory-safe code. 

**4. Public Sector Resistance to Closed Ecosystems**
A coalition of developers launched an open letter (keepthingsopen.com) demanding NHS England maintain its codebase as open-source. *Why it matters:* Following yesterday's backlash against Microsoft and Anthropic's closed-ecosystem tactics, public sector and healthcare developers are pushing back. The tension between proprietary, highly capable enterprise AI ecosystems and transparent, auditable public infrastructure will likely result in divergent tech stacks over the next 5 years.

**5. "The Gay Jailbreak" Proves LLM Vulnerabilities Remain Unsolved**
A new prompt-injection framework dubbed "The Gay Jailbreak" (via ZetaLib) gained traction on GitHub today, showcasing novel methods to bypass LLM safety protocols. *Why it matters:* As AI assistants gain deeper access to local file systems and enterprise codebases (like Claude Code), the proliferation of sophisticated prompt injection techniques poses a severe threat. This guarantees a massive upcoming surge in specialized AI-application security tooling over the next 3-5 years.

**6. Claude Code's Plugin Ecosystem Expands with "Destiny"**
A developer released *Destiny*, a classical East Asian astrology plugin for Anthropic's Claude Code that uses a deterministic Python script to compute birth charts while leveraging Claude for narrative prose. *Why it matters:* Despite Anthropic's recent heavy-handed restrictions on mentioning competitor tools, the community is enthusiastically building extensible, local plugins for Claude Code. This signals a desire for AI assistants to act as extensible terminal operating systems rather than rigid corporate chat interfaces.
